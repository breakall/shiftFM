# shiftFM

## Hardware

Tested with:
* RTL-SDR v3
* raspberry pi 3b+ running raspbian 10 headless


## Pre-reqs

* Osmocom RTL-SDR drivers
* rtl_fm
* ffmpeg


## Usage
parameters: [frequency in MHz] [recording duration in seconds] [name of program]
# example: python3 fmshift.py 96.1 News---> News_10-04-2020.mp3
